# Agent OS V6.0 -- Project Overview

## Product Introduction

Agent OS is a Capital Grade AI Agent Economy Platform -- a multi-region, multi-tenant, event-driven operating system for the AI agent economy. It enables organizations to create, manage, deploy, and monetize AI agents in a marketplace ecosystem.

## What Problem Does It Solve

Agent fragmentation: Organizations run multiple AI agents without unified management, billing, or monitoring
Lack of marketplace: No standard way to publish, discover, and purchase AI agents
Enterprise requirements: Multi-tenancy, RBAC, audit logging, rate limiting needed for production

## Target Users

Platform Administrators: Manage tenants, users, system health, global settings
Agent Developers: Create, configure, test, and publish AI agents
Business Users: Browse marketplace, purchase agents, run workflows
Tenant Admins: Manage users within their organization, monitor usage

## System Modules

Auth: JWT login, RBAC, tenant/user management
Dashboard: System health, service grid, quick stats
Agent Studio: Agent CRUD, list, detail
Workflow Studio: Workflow CRUD, DAG designer placeholder
Marketplace: Agent discovery, categories, pricing
Billing: Credit balance, usage overview
Files: File upload, list, delete
Models: Model list, connection test
Tasks: Task list, detail, execution logs
Settings: System name, model, timezone, log level
